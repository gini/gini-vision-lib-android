createIntentForImportedFile

Screen API

When your application receives a file from another application you can use this method to create an Intent for launching the Gini Vision Library.

Start the Intent with startActivityForResult to receive the GiniVisionError in case there was an error.

Return

an Intent for launching the Gini Vision Library

Parameters

analysisActivityClass

the class of your application's AnalysisActivity subclass

context

Android context

intent

the Intent your app received

reviewActivityClass

the class of your application's ReviewActivity subclass
